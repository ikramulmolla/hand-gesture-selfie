# Hand Gesture Selfie
A Python project that allows users to take selfies using hand gestures.

## Requirements
- Python 3.x
- OpenCV
- MediaPipe
- pyttsx3
- tkinter

## How to Run
1. Install the required dependencies:
   ```bash
   pip install opencv-python mediapipe pyttsx3 tkinter
2. Run the script:
    ```bash
   python HandGestureSelfie.py
